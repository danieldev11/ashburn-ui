# Ashburn/UI Sections

Pre-built section blocks for landing pages and marketing sites.

## Components

### HeroPrimary
Main hero section with headline, subheadline, CTAs, and optional background slot.
- Supports Tier 3 motion for entrance.

### ServicesGrid
Grid of service cards with icons.
- Uses `StaggerList` for revealing cards.

### ProcessSteps
Timeline-style process steps.
- Alternating layout on desktop.
- Numbered steps.

### TestimonialMarquee
Infinite scrolling marquee of testimonials.
- Two rows moving in opposite directions.
- Tier 3 continuous motion.

### PricingTiers
Pricing tables with feature lists.
- Highlights "Most Popular" tier.

### ContactSection
Simple contact form card.
- Standard fields: Name, Email, Message.

### DashboardShell
Layout shell for dashboard applications.
- Responsive sidebar (Sheet on mobile, fixed on desktop).
- Navigation links support icons.
